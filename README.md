# Pathuri Sujith Reddy - Personal Portfolio

A modern, responsive, and dynamic personal portfolio website developed to showcase my education, technical skills, projects, certifications, resume, and professional information.

The portfolio also includes a secure Admin Dashboard that allows portfolio content to be managed dynamically using Firebase.

---

## About Me

I am **Pathuri Sujith Reddy**, a Computer Science Engineering student and aspiring Full Stack Web Developer.

I am interested in building modern, responsive, practical, and user-focused web applications using frontend and backend technologies.

---

## Features

### Portfolio Website

- Modern and responsive user interface
- Hero section with professional introduction
- About Me section
- Education details
- Technical skills with proficiency levels
- Featured projects
- Certificates and certifications
- Resume viewing and management
- Contact information
- Contact message form
- GitHub profile integration
- LinkedIn profile integration
- LeetCode profile integration
- HackerRank profile integration
- Responsive navigation
- Scroll-to-top functionality

### Admin Dashboard

The portfolio includes a dedicated Admin Dashboard for managing portfolio content.

Admin features include:

- Secure Firebase Authentication
- Manage Profile
- Manage About section
- Manage Education
- Add and delete Skills
- Add, edit, and delete Projects
- Manage Certificates
- Manage Certifications
- Manage Resume
- Manage Contact Information
- Manage Social Links
- View Contact Messages
- Delete Messages
- Dynamic Dashboard statistics
- Real-time portfolio updates

---

## Firebase Integration

The portfolio uses Firebase for cloud-based data management.

### Firebase Services Used

- Firebase Authentication
- Cloud Firestore
- Firebase Storage integration setup

Portfolio information stored in Firestore can be accessed across different browsers and devices.

Changes made through the Admin Dashboard are synchronized with the public portfolio.
